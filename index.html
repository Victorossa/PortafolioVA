ERP New POS

Fecha 01/07/2017 Puedes verlo en: (Aplicacion de Escritorio)

Desarrollo de aplicación administrativa para la gestión comercial llamada NEW POS, se realizó la planeación, análisis, diseño, programación, pruebas he implementación. Se hace soporte y se instaló toda la red de hardware.

New POS fue el resultado de 1 año de trabajo para crear en una aplicación todo el funcionamiento y control de un negocio dedicado a productos de ferreteria. Desde la creacion y de productos y su control hasta facturacion, control de inventario y fidelización de clientes, este proyecto ha sido la base para la toma de desiciones importantes en la organizacion.

Tecnologias Aplicadas

  • Visual Basic
  • Sql Server
  • Reporting Services
  • Power Pivot (BI)
